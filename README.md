# 🌾 Tribute Page – Dr. Norman Borlaug

A responsive tribute page dedicated to **Dr. Norman Borlaug**, the Nobel Peace Prize winner whose agricultural innovations helped save millions from hunger around the world.

This project was built as part of the **freeCodeCamp Responsive Web Design Certification** to practice semantic HTML, responsive design, and CSS styling.

## 📖 About Dr. Norman Borlaug

Dr. Norman Borlaug was an American agronomist and humanitarian whose work in developing high-yield, disease-resistant wheat varieties sparked the **Green Revolution**, dramatically increasing food production and reducing global hunger.

Learn more here:

https://en.wikipedia.org/wiki/Norman_Borlaug
